# LogiTask: Sistema Ágil de Gestão Logística

Projeto desenvolvido para a **TechFlow Solutions**, focado no gerenciamento de tarefas e fluxos de trabalho para uma startup de logística.

## Escopo do Projeto
O sistema permite o controle completo (CRUD) de entregas e tarefas, incluindo título, prioridade, prazos e status. O objetivo é oferecer visibilidade em tempo real para a operação logística.

## Funcionalidades Implementadas
- **Criar Tarefas:** Registro de novas entregas com definição de prioridade.
- **Listar Entregas:** Visualização completa de todos os itens no sistema.
- **Atualizar Status:** Movimentação de tarefas entre as fases (A Fazer, Em Progresso, Concluído).
- **Excluir Tarefas:** Remoção de registros via console por ID único.

## Metodologia e Gestão
- **Metodologia Ágil:** Utilizamos **Kanban** via GitHub Projects para monitorar o fluxo.
- **Gestão de Mudanças:** Durante o desenvolvimento, o escopo foi alterado para incluir campos de **Prioridade** e **Prazo**, atendendo a uma nova demanda de urgência do cliente.
- **Qualidade (CI/CD):** Pipeline configurado via **GitHub Actions** para executar testes unitários automaticamente a cada commit.

## Como Executar
1. Certifique-se de ter o Python 3 instalado.
2. No terminal, execute: `python main.py`

3. Para rodar os testes: `python test_main.py`
